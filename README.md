# 📊 Excel Sales Dashboard

## 🔹 Project Overview

This project analyzes sales data and presents insights using an interactive Excel dashboard.

## 🔹 Features

* Dynamic KPIs (Revenue, Profit, Units Sold, Avg Daily Revenue)
* Interactive slicers (Region, Product, Salesperson)
* Visual charts for analysis

## 🔹 Key Insights

* Total revenue is approximately ₹11 Cr
* Top-performing product contributes most sales
* Regional performance varies significantly
